# AGENTS.md

Welcome, Agent! Here are some important instructions for this repository:

1. This is `libais`, a C++ and Python library for decoding maritime Automatic Identification System messages.
2. The Python library is built with C++ extensions using `setup.py`.
3. Please ensure tests pass by running `pytest` when modifying code.
4. Python files are located in `ais/`, tests in `test/`, and C++ source in `src/`.
